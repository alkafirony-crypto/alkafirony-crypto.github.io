## GitHub Pages upload

Upload the **contents of this folder** to the root of the GitHub Pages repository:

- `index.html`
- `.nojekyll`
- `LICENSE`
- `README.md`
- `assets/`

Do not upload only `index.html`, because this MIT template uses local CSS, JavaScript, and image assets.
